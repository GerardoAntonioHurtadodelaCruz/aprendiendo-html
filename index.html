¿Que es HTML?

Html es un estandar para la creación de paginas web, describe su estructura y consiste de una serie de elementos llamados etiquetas, estas le indican al navegador como mostrar el contenido en pantalla.

  1. HTML son las siglas de Hyper Text Markup Lenguage
  2. HTML es el lenguaje de marcado estándar para crear páginas web
  3. HTML describe la estructura de una página web
  4. HTML consta de una seria de elementos
  5. Los elementos HTML le dicen al navegador cómo mostrar el contenido
  6. Los elementos HTML etiquetan partes de contenido como "esto es un encabezado" o "esto es un parrafo"

Ejemplo de la estructura basica:

Doctype
-Este elemento nos permite inidicar el tipo de archivo
html
-Indica que el documento es tipo html
head
-Inidica que es la cabezera del documento
title
-Indica el titulo de la pagina que siempre se encuentra dentro de la cabezera
