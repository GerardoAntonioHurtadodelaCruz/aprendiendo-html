Etiquetas Basicas

Etiqueta Significado
<!DOCTYPE> Define el tipo de documento que se maneja en html
<html> Define un documento html
<head> Contiene meta data e Información de la pagina como tal
<title> Define el titulo para la pagina, este va a aparecer en la cabezera de la pagina
<body> Define el cuerpo del documento, osea que aqui va a ir todo el codigo que se muestra en el navegador
<h1> a <h6> Se conocen como headers, sirven para poner titlos, entre más pequeño el número es más importante el titulo y más grande la fuente
<p> Esto se usa para hacer un parrafo y escribir texto simple adentro
<br> Inserta un quiebre de linea, osea un espacio en blanco para poder separar elementos
<hr> Define un cambio de contenido, esto va a dar un espacio de separación entre elementos y tambien pinta una linea horizontal
<!--...--> Con esto se puede generar un comentario en cualquier parte del documento html
