<textarea></textarea>

La etiqueta <textarea> define un control de entrada de texto de varis líneas

El elemento <textarea> se usa a menudo en un formulario para recopilar entradas de usuario como comentarios o reseñas

Un texarea puede contener un número ilimitado de caracteres y el texto representa una fuente de ancho fijo

El tamaño de un textarea se especifica mediante los atributos cols y rows(o con CSS)
