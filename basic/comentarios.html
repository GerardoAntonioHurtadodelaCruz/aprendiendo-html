La etiqueta de comentario se utiliza para insertar comentarios en el código fuente. Los comentarios
no se muestran en los navegadores

Puede usar comentarios para explicar su código, lo que puede ayudarlo cuando edite el código fuente en
una fecha posterior. Esto es especialmente útil si tiene mucho código

NOTA: Puede utilizar la etiqueta de comentarios para "ocultar" los scripts de los navegadores que
no sean compatibles con los scripts(para que no los muestren como texto sin formato)

NOTA: Las dos barras diagonales (//) es el símbolo de comentario de JavaScript

Atributos

La etiqueta de comentario no admite ningún atributo
