<address></address>

La etiqueta <address> define la información de contacto del autor/propietario de un documento o artículo

La información de contacto puede ser una dirección de correo electróinico, URL, dirección física, número de
teléfono, identificador de redes sociales, etc.

El texto en el elemento <address> generalmente se muestra en cursiva, y los navegadores siempre agregarán
un salto de línea antes y después del elemento <address>

Atributos

Todas las etiquetas pueden poseer algun atributo, a continuación se mostraran los correspondientes a esta etiqueta:

Atributo Tipo de Atributo Descripción
accesskey Global Especifica una tecla de método abreviado para activar/ enfocar un elemento
class Global Especifica uno o más nombre de clase para un elemento
contenteditable Global Especifica si el contenido de un elemento es editable o no
data-* Global Se utiliza para almacenar datos personalizados privados de la página
dir Global Especifica la dirección del texto del contenido de un elemento
draggable Global Especifica si un elemento se puede arrastrar o no
hidden Global Especifica que un elemento aún no es o ya no es relevante
id Global Especifica una identificación única para un elemento
lang Global Especifica el idioma del contenido del elemento
spellcheck Global Especifica si se revisará la ortografía y la grmática del elemento o no
style Global Especifica un estilo CSS en línea para un elemento
tabindex Global Especifica el orden de tabulación de un elemento
title Global Especifica información adicional sobre un elemento
translate Global Especifica si el contenido de un elemento debe traducirse o no
xmlns Individual Especifica el atributo del espacio de nombres XML
