¿Que es un elemento HTML?

Un elemento HTML esta definido por una etiqueta de inicio, el contenido y una etiqueta final

NOTA: Algunos elementos de html no tienen etiqueta de cierre.

<etiquetainicio> Contenido </etiquetafinal>

Elementos Anidados

En HTML los elementos o etiquetas se pueden "anidar", esto quiere decir que los elementos pueden contener otros elementos dentro de si para formar uno nuevo.
